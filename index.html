Claude Code Mastery

From Zero to Autonomous Agent Teams — a structured, hands-on guide for builders who learn by doing.

This guide takes you from "what even is a terminal?" to running fully autonomous agent teams that build software in parallel — without requiring a computer science degree. Each level introduces one core concept, explains it with intuition first, and ends with a checkpoint project that proves you're ready to move on.

How to use this guide: Go in order. Don't skip levels. Every checkpoint project is designed so that completing it means you genuinely understand the material — not just that you copied commands.

  • 01 The Terminal: Your New Home Base
    What a shell is, navigating the filesystem, PATH, environment variables, and the differences between shells.
  • 02 How Claude Code Actually Works
    The tool-use loop, the context window, how Claude reads and writes files, and what happens when you press Enter.
  • 03 File Systems and Project Structure
    Directory trees, naming conventions, common project layouts, and why organization matters at scale.
Practitioner Tier
  • 04 CLAUDE.md: The Most Important File You'll Ever Write
    What CLAUDE.md is, where it lives, how Claude reads it, and how to write one that transforms your results.
  • 05 Prompt Engineering for Claude Code
    How prompting in Claude Code differs from chat, plan mode, task decomposition, and the art of iteration.
  • 06 Git: Your Safety Net and Time Machine
    Version control fundamentals, branches, commits, recovering from mistakes, and how Claude Code uses git.
  • 07 Reading Code Without Writing It
    Understanding programming languages conceptually, reading code as structured English, and reviewing Claude's output.
Intermediate Tier
  • 08 Configuration and Customization Deep Dive
    settings.json, environment variables, permission modes, model selection, custom slash commands, and the .claude/ directory.
  • 09 APIs: How Software Talks to Software
    What an API is, REST basics, authentication, JSON, the Anthropic API, and making Claude Code call external services.
  • 10 MCP: Giving Claude Code Superpowers
    Model Context Protocol explained, installing MCP servers, practical integrations, and building your own server.
  • 11 Hooks: Automation at Every Step
    Lifecycle event listeners, hook types, auto-formatting, quality gates, and when hooks help vs. hurt.
Advanced Tier
  • 12 Subagents: Divide and Conquer
    Isolated Claude instances, the Task tool, context isolation, scoping work, and practical delegation patterns.
  • 13 Headless Mode and Scripting
    Running Claude Code non-interactively, piping I/O, automated pipelines, and scheduled execution.
  • 14 The Claude Code SDK
    Programmatic agent spawning, streaming, conversation management, and building custom tooling.
Mastery Tier
  • 15 Agent Teams: Parallel Minds
    Lead + teammates, shared task lists, inter-agent communication, team hooks, and cost management.
  • 16 Building Fully Autonomous Coding Systems
    Combining everything: CLAUDE.md, hooks, subagents, teams, MCP, and headless execution into end-to-end autonomy.
  • 17 The Frontier: What's Next
    Emerging patterns, staying current, the discipline of managing AI workers, and building your own playbook.

Built for builders who learn by doing. No CS degree required.
